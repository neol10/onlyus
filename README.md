# OnlyUs — Frontend scaffold

Scaffold inicial do aplicativo OnlyUs (Next.js + Tailwind + Firebase).

Instalação:

```bash
npm install
```

Rodar em desenvolvimento:

```bash
npm run dev
```

Pontos importantes:
- Atualize `src/firebase/firebaseClient.js` com as credenciais reais do Firebase.
- Para PWA completa, considere usar `next-pwa` e gerar ícones apropriados.
